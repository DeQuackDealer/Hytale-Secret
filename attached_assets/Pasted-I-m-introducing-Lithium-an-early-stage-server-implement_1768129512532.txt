Lithium is not a finished server and will not replace the official Java server. The project is currently focused on building a modern, high-performance foundation for what will become a full server platform.

A core goal of Lithium is to allow server developers to build mods and plugins in C# instead of Java, using the same language, tools, and type system as the game/server itself.

üöÄ Why C#?
The long-term goal is to explore how far a modern C# stack can be pushed for a large-scale game server. Lithium is built around low-level C# features that are well-suited for this type of workload:

SIMD: Hardware-accelerated math for physics, collisions, and world generation.
Unsafe code & pointers: Used selectively to control memory and optimize critical paths.
Zero-allocation networking: Span<T> and Memory<T> are used to process packets with minimal GC pressure.
JIT & PGO: Profile-guided optimizations allow hot code paths to get faster over time.

These are the building blocks of the engine that is being developed right now.
